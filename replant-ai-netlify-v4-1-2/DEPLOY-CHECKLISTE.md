# RePlant Deploy-Checkliste

## GitHub

- [ ] `package.json` liegt auf der obersten Ebene
- [ ] `netlify.toml` liegt auf der obersten Ebene
- [ ] `README.md` liegt auf der obersten Ebene
- [ ] `START-HIER.md` liegt auf der obersten Ebene
- [ ] `public/` wurde vollständig hochgeladen
- [ ] `netlify/` wurde vollständig hochgeladen
- [ ] `scripts/` wurde vollständig hochgeladen
- [ ] `tests/` wurde vollständig hochgeladen

## Netlify

- [ ] Build command: `npm run verify`
- [ ] Publish directory: `public`
- [ ] Functions directory: `netlify/functions`
- [ ] erster Deploy erfolgreich
- [ ] Website im Grundmodus getestet

## AI

- [ ] `XAI_API_KEY` als geheime Netlify-Variable gespeichert
- [ ] nach dem Speichern erneut deployed
- [ ] AI-Schalter neben der Hauptaktion sichtbar
- [ ] AI-Schalter lässt sich aktivieren
- [ ] AI-Ergebnis zeigt `RePlant AI`

## Ergebnisdarstellung

- [ ] Produktersetzungen stehen unter **Ersetzte Produkte**
- [ ] Portionsänderungen stehen unter **Angepasste Mengen**
- [ ] keine zusätzliche AI-Karte rechts
- [ ] kein zusätzlicher AI-Status oben rechts
- [ ] kein sichtbarer Erfolgsstatus unter dem Hauptbutton
